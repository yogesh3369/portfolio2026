# Mainframe - Creative Agency Landing Page

A full-screen hero landing page for Mainframe creative agency featuring:

- **Mouse-scrub controlled background video** - Video scrubs forward/backward based on horizontal mouse movement
- **Custom typewriter effect** - Animated text reveal with blinking cursor
- **Responsive navbar** - Desktop navigation with mobile hamburger menu
- **Action pill buttons** - Interactive CTAs with fade-in animation
- **Custom fonts** - HelveticaNowDisplay-Medium and HelveticaNowDisplayW01-Rg

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

- Full-screen background video with mouse-scrub interaction
- Responsive design (mobile-first approach)
- Custom typewriter hook with configurable speed and delay
- Smooth transitions and hover effects
- Email copy-to-clipboard functionality
